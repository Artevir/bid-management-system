将投标文档相关的功能整合到导航栏的"标书文档"分类下，提供更清晰的功能组织结构。

## 📊 功能整合说明

### 新整合的功能

| 功能 | 路由 | 说明 |
|------|------|------|
| 文档列表 | `/bid` | 查看所有投标文档 |
| 购买招标文件 | `/bid-document-purchases` | 从投标事务移入 |
| 文档解读 | `/bid/interpretations` | 关联招标文件解读 |
| 文档模板 | `/bid/templates` | 管理文档模板 |
| 审批流程 | `/bid/approval` | 管理文档审批 |
| 盖章安排 | `/bid-seal-applications` | 从投标事务移入 |
| 签章管理 | `/bid/seal` | 文档签章申请 |
| 文档统计 | `/bid/statistics` | 查看文档统计数据 |

### 整合原因

1. **业务逻辑**: 购买招标文件和盖章安排本质上是投标文档管理的一部分
2. **用户体验**: 将文档相关功能集中管理，减少菜单层级
3. **功能关联**: 这些功能都与投标文档的生命周期紧密相关
