# 投标管理平台系统技术文档

## 一、系统概述

投标管理平台是一个企业级的投标全流程管理系统，支持从项目创建、文档编制、团队协作、审核签发、导出投标、打印安排、盖章申请、去投标、领取中标通知书、履约保证金、签订书面合同等完整业务流程。系统采用 AI-native 架构，集成智能编标、智能审核、智能报价等 AI 能力。

## 三、系统架构

### 3.1 整体架构

```
┌─────────────────────────────────────────────────────────────────┐
│                         前端层 (Next.js)                         │
├─────────────────────────────────────────────────────────────────┤
│  App Router  │  React Components  │  shadcn/ui  │  TanStack    │
├─────────────────────────────────────────────────────────────────┤
│                         API 层 (Route Handlers)                  │
├─────────────────────────────────────────────────────────────────┤
│  认证授权  │  业务服务  │  文件处理  │  AI 集成  │  工作流引擎  │
├─────────────────────────────────────────────────────────────────┤
│                         数据层 (PostgreSQL + Drizzle ORM)        │
├─────────────────────────────────────────────────────────────────┤
│                         外部服务集成                              │
│  LLM API  │  文件解析  │  Web搜索  │  对象存储  │  电子签章     │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 目录结构

```
src/
├── app/                    # Next.js App Router
│   ├── (app)/             # 认证后的应用页面
│   ├── admin/             # 后台管理页面
│   ├── api/               # API 路由 (262个接口)
│   └── ...                # 其他页面
├── components/            # React 组件
│   └── ui/               # shadcn/ui 组件
├── db/                    # 数据库
│   ├── schema.ts         # 主表定义 (109张表)
│   └── *-schema.ts       # 分模块表定义
├── lib/                   # 业务逻辑层
│   ├── ai-governance/    # AI 治理
│   ├── bid/              # 标书业务

│   ├── company-sync/     # 公司信息联动
│   ├── platform-sync/    # 政采单位联动
│   └── ...               # 其他服务

```

## 七、核心业务流程

### 7.1 项目全生命周期

```
项目创建 → 文件解读 → 文档编制 → 审核签发 → 导出投标 → 打印安排 
    → 盖章申请 → 去投标 → 开标 → 领取中标通知书 → 履约保证金 → 签订合同 → 项目归档
```

### 7.2 数据联动架构

```
┌─────────────────────────────────────────────────────────────┐
│                    文件解读 (bidDocumentInterpretations)     │
│                              ↓                               │
│                自动提取：招标单位、代理机构、时间节点           │
│                              ↓                               │
│              智能匹配政采单位 (platformId/agentPlatformId)    │
└─────────────────────────────────────────────────────────────┘
                               ↓
┌─────────────────────────────────────────────────────────────┐
│                     公司信息联动 (companies)                  │
│                              ↓                               │
│          自动填充：对接人、资质文件、文档框架、地址信息         │
└─────────────────────────────────────────────────────────────┘
                               ↓
┌─────────────────────────────────────────────────────────────┐
│                     业务模块自动获取联动信息                   │
├─────────────────────────────────────────────────────────────┤
│ • 买标书：政采单位地址、联系方式、在线报名支持                 │
│ • 打印安排：打印店位置、交付时间                              │
│ • 盖章申请：友司地址、盖章要求                                │
│ • 去投标：投标地址、导航坐标、是否支持在线投标                 │
│ • 开标现场：开标地点、是否支持直播                            │
│ • 领取通知书：领取地址、携带材料清单                          │
│ • 履约保证金：收款账户、缴纳流程                              │
│ • 签订合同：合同模板、条款一致性检查                          │
└─────────────────────────────────────────────────────────────┘
```
