## 系统概述

投标管理平台是一个企业级投标全流程管理系统，支持项目创建、文档编制、团队协作、审核签发、导出投标等完整业务流程。

### 系统架构

- **前端**: Next.js 15 + React 19 + TypeScript
- **后端**: Next.js API Routes
- **数据库**: PostgreSQL
- **文件存储**: 对象存储 (S3兼容)
- **实时通信**: WebSocket

### 核心模块

1. **项目管理** - 项目创建、进度跟踪、团队协作
2. **文档管理** - 文件上传、在线编辑、版本控制
3. **智能编标** - AI辅助文档编制、模板管理
4. **审核流程** - 四级审核流、批注管理
5. **知识库** - 企业投标知识沉淀与复用
6. **报表统计** - 审核效率、项目分析

## 组织管理

### 创建组织

1. 进入「系统管理」→「组织管理」
2. 点击「新建组织」
3. 填写组织信息：
   - 组织名称
   - 组织代码
   - 联系人
   - 联系电话
   - 地址
4. 保存后系统自动创建管理员账号

### 组织配置

- **存储配额**: 设置组织的文件存储上限
- **用户配额**: 设置组织的用户数量上限
- **功能权限**: 开通/关闭特定功能模块
- **有效期**: 设置组织账号有效期

## 用户管理

### 创建用户

1. 进入「系统管理」→「用户管理」
2. 点击「新建用户」
3. 填写用户信息：
   - 用户名（登录账号）
   - 姓名
   - 邮箱
   - 手机号
   - 所属组织
   - 角色
4. 系统自动发送账号激活邮件

### 用户状态管理

- **启用**: 用户可正常登录使用
- **禁用**: 用户无法登录，数据保留
- **删除**: 用户账号删除，数据转移或删除

### 重置密码

1. 在用户列表找到目标用户
2. 点击「重置密码」
3. 选择重置方式：
   - 发送重置邮件
   - 设置临时密码

## 角色与权限管理

### 预置角色

| 角色 | 权限范围 |
|------|----------|
| 超级管理员 | 系统全部权限 |
| 组织管理员 | 组织管理权限 |
| 项目经理 | 项目管理权限 |
| 编标人员 | 文档编辑权限 |
| 审核人员 | 审核权限 |
| 只读用户 | 仅查看权限 |

### 自定义角色

1. 进入「系统管理」→「角色管理」
2. 点击「新建角色」
3. 配置角色权限：
   - 模块权限（项目、文档、审核等）
   - 操作权限（创建、编辑、删除等）
   - 数据权限（全部、本部门、本人）

### 权限配置建议

- 编标人员：分配「文档编辑」「模板使用」权限
- 审核人员：分配「审核」「批注」权限
- 项目经理：分配「项目管理」「团队分配」权限

## 项目管理

### 项目创建

1. 进入「项目管理」→「项目列表」
2. 点击「新建项目」
3. 填写项目信息：
   - 项目名称
   - 招标编号
   - 招标方
   - 截标时间
   - 项目类型
   - 预算金额
4. 分配项目成员

### 项目配置

- **审核流程**: 选择四级审核流程模板
- **存储空间**: 设置项目文件存储配额
- **权限设置**: 配置项目成员权限
- **通知设置**: 配置项目通知规则

### 项目状态管理

| 状态 | 说明 |
|------|------|
| 筹备中 | 项目准备阶段 |
| 编制中 | 文档编制阶段 |
| 审核中 | 审核流程进行中 |
| 已完成 | 投标已完成 |
| 已归档 | 项目已归档 |

## 审核流程配置

### 四级审核流程

1. **编制审核** - 编制人自检
2. **部门审核** - 部门负责人审核
3. **专业审核** - 专业人员审核
4. **领导签发** - 领导最终签发

### 审核节点配置

1. 进入「系统管理」→「审核配置」
2. 选择审核模板
3. 配置审核节点：
   - 节点名称
   - 审核人/审核角色
   - 审核时限
   - 退回规则
   - 通知配置

### 审核人分配

- **按角色分配**: 指定角色，该角色用户均可审核
- **按人员分配**: 指定具体审核人
- **自动分配**: 系统根据规则自动分配

## 知识库管理

### 知识库结构

```
知识库
├── 公司资质
│   ├── 营业执照
│   ├── 资质证书
│   └── 荣誉证书
├── 人员信息
│   ├── 人员简历
│   └── 资格证书
├── 业绩案例
│   ├── 项目案例
│   └── 客户评价
└── 技术方案
    ├── 技术文档
    └── 解决方案
```

### 知识录入

1. 进入「知识库管理」
2. 选择分类
3. 点击「添加知识」
4. 填写知识信息：
   - 标题
   - 内容/附件
   - 标签
   - 有效期

### 知识审核

- 新增知识需审核后才能使用
- 管理员可设置免审核规则
- 过期知识自动标记

## 数据备份与恢复

### 自动备份

- 系统每日自动备份数据库
- 备份文件保留30天
- 备份文件存储在对象存储

### 手动备份

1. 进入「系统管理」→「数据备份」
2. 点击「立即备份」
3. 选择备份范围：
   - 数据库
   - 文件存储
   - 配置信息

### 数据恢复

1. 进入「系统管理」→「数据备份」
2. 选择备份记录
3. 点击「恢复」
4. 确认恢复操作

> ⚠️ 恢复操作会覆盖当前数据，请谨慎操作！

## 常见问题处理

### 用户无法登录

1. 检查用户状态是否启用
2. 检查密码是否正确
3. 检查账号是否锁定
4. 重置密码后重试

### 文件上传失败

1. 检查存储空间是否充足
2. 检查文件大小是否超限
3. 检查文件格式是否支持
4. 检查网络连接状态

### 审核流程卡住

1. 检查审核人是否正确分配
2. 检查审核人账号状态
3. 手动重新分配审核人
4. 管理员强制推进流程

### 数据显示异常

1. 清除浏览器缓存
2. 检查数据库连接
3. 查看错误日志
4. 联系技术支持

### 系统响应缓慢

1. 检查系统资源使用
2. 检查数据库性能
3. 检查网络带宽
4. 必要时重启服务
