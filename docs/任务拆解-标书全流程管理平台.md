## 📋 项目概览

### 核心目标
- **效率提升**：标书制作周期缩短 60% 以上，减少 80% 重复劳动
- **质量管控**：废标风险降低 80% 以上，提升投标成功率
- **协同优化**：部门对接效率提升 70%
- **知识沉淀**：形成可复用的投标知识库
- **保密合规**：构建全流程保密管控体系
- **数字化转型**：投标业务全流程数字化管理

### 技术栈
| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 前端 | Vue.js + Element UI | 国内主流，组件丰富，易于上手 |
| 后端 | Spring Boot + MyBatis Plus | 企业级首选，生态完善 |
| AI框架 | TensorFlow/PyTorch | 主流AI框架，支持本地部署 |
| 数据库 | PostgreSQL + Redis | 开源稳定，支持加密，性能优异 |
| 文件存储 | MinIO | 开源高性能，支持本地部署 |
| 加密技术 | AES-256 + RSA | 企业级加密标准 |
| 部署环境 | Linux本地服务器 | 安全性高，符合保密要求 |
