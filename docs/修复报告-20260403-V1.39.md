# 修复报告-20260403-V1.39

**版本**: V1.39  
**日期**: 2026-04-03

## 问题描述

构建失败，报错信息如下：

```
Type error: Object literal may only specify known properties, and 'bold' does not exist in type 'IParagraphOptions'.
```

## 问题原因

在 V1.37 版本中，使用 docx 库生成 Word 文档时，直接在 `Paragraph` 对象中使用了 `bold: true` 属性。但实际上 docx 库的 `Paragraph` 类型并不支持 `bold` 属性，需要使用 `TextRun` 组件来设置文本样式。

## 修复内容

**文件**: `src/app/api/interpretations/[id]/export/route.ts`

将所有错误的写法：
```typescript
new Paragraph({ text: '标题', bold: true })
```

修改为正确的写法：
```typescript
new Paragraph({ children: [new TextRun({ text: '标题', bold: true })] })
```

修改涉及以下位置：
- 基本信息表格的表头（第130行）
- 技术规格表格的表头（第152-154行）
- 评分细则表格的表头（第184-187行）
- 核对清单表格的表头（第218-220行）
