# 修复报告-20260403-V1.40

**版本**: V1.40  
**日期**: 2026-04-03

## 问题描述

构建失败，报错信息如下：

```
Type error: Argument of type 'Table' is not assignable to parameter of type 'Paragraph'.
```

## 问题原因

在 V1.39 版本修复中，将 `generateWord` 函数的 `children` 数组类型定义为 `Paragraph[]`，但实际上 Word 文档的 section children 既可以包含 `Paragraph` 也可以包含 `Table`，类型定义不完整导致 `Table` 无法 push 到数组中。

## 修复内容

**文件**: `src/app/api/interpretations/[id]/export/route.ts`

修改前：
```typescript
const children: Paragraph[] = [];
```

修改后：
```typescript
const children: (Paragraph | Table)[] = [];
```
