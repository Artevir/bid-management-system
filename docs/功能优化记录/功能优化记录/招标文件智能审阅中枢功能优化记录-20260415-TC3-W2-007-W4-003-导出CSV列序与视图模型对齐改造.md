# 招标文件智能审阅中枢功能优化记录-20260415-TC3-W2-007-W4-003-导出CSV列序与视图模型对齐改造

## 1. 本次目标

- 面向 `TC3-W2-007` 与 `TC3-W4-003` 的交叉缺口，补齐导出能力的“列名、顺序、口径”约束。
- 把中枢导出从“仅 JSON 视图”升级为“可下载 CSV 且列顺序固定”的可交付功能。

## 2. 本次代码改动

### 2.1 新增导出视图与 CSV 工具

- 新增 `src/app/api/tender-center/_export.ts`
  - 固化 10 类导出视图代码（`*_export_view`）
  - 固化每个视图的 CSV 列顺序定义
  - 新增 CSV 生成器（含逗号/引号/换行转义）
  - 新增视图代码合法性校验方法

### 2.2 升级导出接口能力

- 修改 `src/app/api/tender-center/projects/[projectId]/versions/[versionId]/export/route.ts`
  - 支持 `format=csv&view=<export_view_code>` 导出单视图 CSV
  - 默认保留 `json` 行为，向后兼容
  - 新增/规范以下视图模型，避免直接回传底层表行：
    - `scoring_export_view`
    - `technical_export_view`
    - `framework_export_view`
  - CSV 返回含 `Content-Disposition` 与 UTF-8 BOM，便于表格工具直接打开

### 2.3 新增单测矩阵

- 新增 `tests/unit/tender-center-export-csv.test.ts`
  - 导出视图 code 校验
  - 列顺序稳定性校验
  - CSV 构建顺序校验
  - CSV 特殊字符转义校验
