## 一、如何使用本计划（逐项核对规则）

1. **每一项只有两种终态**: `□ 未开始` → `☑ 已交付且验收通过`（禁止「基本完成」）。
2. **每一项必须可客观判定**: 表格中「验收标准」为**二元**或可脚本化断言（命令、HTTP 状态、DB 行数、截图编号、用例 ID）。
3. **依赖不可跳**: 「依赖」列所列 ID 全部 `☑` 后，本项才可进入验收。
4. **缺陷门禁**: 见 **§三**；未达门禁不得将波次标记为完成。
5. **追溯**: 「依据」列填写 **总装稿编号 + 章节/表名**（若只有 Word 内表号，则写「见 Word 某表」并在 PR 中附对照截图）。

**任务 ID 规则**: `TC3-<波次>-<序号>`，例如 `TC3-W4-012`。同一 ID 在全仓库唯一。
