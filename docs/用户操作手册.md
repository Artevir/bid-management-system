## 系统概述

### 平台定位

标书全流程管理平台是一款面向企业投标业务的AI原生生产平台，提供从招标文件解析到标书提交的全流程数字化管理。

### 核心能力

- **📄 文档智能接入**：自动解析招标文件，结构化提取关键信息
- **🎯 投标响应矩阵**：一键生成响应清单，精准匹配招标要求
- **✍️ 智能编标**：AI辅助首稿生成，知识库快速复用
- **🔍 智能审校**：自动检测漏项、错项、格式问题
- **✔️ 多级审核**：四级审核流程，全程可追溯
- **📚 企业知识中台**：历史标书、资质、案例结构化管理

## 功能模块

### 1. 工作台

工作台是您的个人工作中心，展示以下信息：

**统计概览**
- 进行中项目数
- 待处理任务数
- 即将到期节点
- 最近活动记录

**待办事项**
- 待审核标书
- 待处理任务
- 逾期提醒

**快捷入口**
- 创建项目
- 上传文件
- 搜索知识

### 2. 项目管理

#### 2.1 创建项目

1. 点击"新建项目"按钮
2. 填写项目基本信息：
   - 项目名称
   - 招标编号
   - 招标单位
   - 项目类型
   - 关键时间节点
3. 上传招标文件
4. 点击"创建"

#### 2.2 项目详情

项目详情页包含多个标签页：

**驾驶舱**
- 项目进度概览
- 阶段完成情况
- 活动趋势图
- 风险预警
- 里程碑节点

**基本信息**
- 招标信息
- 关键时间节点
- 项目描述
- 项目标签

**文档解析**
- 上传招标文件
- 查看解析进度
- 查看解析结果
- 关键信息提取
- 响应矩阵生成

**项目阶段**
- 查看各阶段进度
- 阶段状态管理
- 阶段时间记录

**关键节点**
- 里程碑管理
- 截止日期提醒
- 完成状态跟踪

**过程记录**
- 会议纪要管理
- 客户对接记录
- 任务分配与跟踪

**项目成员**
- 成员管理
- 权限设置
- 角色分配

#### 2.3 文档解析

1. 进入项目详情 → 文档解析
2. 点击"上传招标文件"
3. 选择PDF/Word文件上传
4. 等待解析完成（通常1-3分钟）
5. 查看解析结果：
   - 时间节点提取
   - 资格要求识别
   - 评分项提取
   - 技术参数提取

### 3. 知识库

#### 3.1 知识分类

知识库按以下维度组织：
- 企业资质
- 业绩案例
- 技术方案
- 人员简历
- 商务条款

#### 3.2 添加知识

1. 进入知识库页面
2. 点击"新建知识"
3. 选择分类
4. 填写标题和内容
5. 添加标签
6. 提交审核

#### 3.3 搜索知识

1. 在搜索框输入关键词
2. 系统支持：
   - 全文搜索
   - 语义搜索
   - 标签筛选
   - 分类筛选

### 4. 模板库

#### 4.1 模板类型

- 技术标模板
- 商务标模板
- 资格标模板

#### 4.2 创建模板

1. 进入模板库
2. 点击"新建模板"
3. 选择模板类型
4. 编辑模板内容
5. 保存发布

#### 4.3 使用模板

1. 选择模板
2. 点击"使用模板"
3. 关联到项目
4. 开始编辑

### 5. 审核中心

#### 5.1 审核流程

平台支持四级审核流程：
1. 制作人自审
2. 部门经理审核
3. 总监审核
4. 总经理审核

#### 5.2 提交审核

1. 完成标书编辑
2. 点击"提交审核"
3. 选择审核流程
4. 填写提交说明
5. 确认提交

#### 5.3 审核操作

审核人可进行以下操作：
- **通过**：流转到下一审核节点
- **驳回**：退回修改
- **转办**：转交他人审核
- **批注**：添加审核意见

### 6. 过程记录

#### 6.1 会议纪要

1. 进入项目 → 过程记录 → 会议纪要
2. 点击"新建会议纪要"
3. 填写会议信息：
   - 会议标题
   - 会议日期
   - 参会人员
   - 会议地点
   - 会议内容
4. 保存记录

#### 6.2 客户对接记录

1. 进入"对接记录"标签
2. 点击"新建记录"
3. 选择对接方式：
   - 电话
   - 邮件
   - 会议
   - 现场考察
4. 记录对接内容
5. 设置后续跟进

#### 6.3 任务管理

1. 进入"任务跟踪"标签
2. 创建任务
3. 分配负责人
4. 设置优先级和截止日期
5. 跟踪任务进度

## 常见问题

### Q1: 忘记密码怎么办？

请联系系统管理员重置密码，或使用登录页面的"忘记密码"功能通过邮箱重置。

### Q2: 无法上传文件？

请检查：
1. 文件格式是否支持（PDF、Word、Excel）
2. 文件大小是否超过限制（默认50MB）
3. 网络连接是否正常

### Q3: 文档解析失败？

可能原因：
1. 文件格式损坏
2. 文件内容为图片（需OCR处理）
3. 服务暂时不可用

建议：重新上传或联系技术支持

### Q4: 如何导出标书？

1. 进入项目详情页
2. 点击"导出"按钮
3. 选择导出格式：
   - Word文档
   - PDF文档
   - 电子投标格式
4. 确认导出

### Q5: 如何管理项目成员权限？

1. 进入项目 → 项目成员
2. 点击"添加成员"
3. 选择用户并设置角色：
   - 项目负责人：全部权限
   - 编辑者：可编辑内容
   - 查看者：仅查看
   - 审核者：可审核

### Q6: 审核流程卡住怎么办？

1. 检查当前审核节点
2. 联系对应审核人
3. 如需加急，可申请催办

### Q7: 如何查看操作历史？

1. 进入系统设置 → 操作日志
2. 可查看所有操作记录
3. 支持按时间、类型筛选
