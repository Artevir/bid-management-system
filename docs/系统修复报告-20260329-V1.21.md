# 系统修复报告-20260329-V1.21

**版本**: V1.21  
**日期**: 2026-03-29  

## 修复背景

服务器执行 `pnpm build` 在 TypeScript 类型校验阶段失败，错误发生在搜索建议 API 调用了 `SearchService` 的私有静态方法。

## Issue 1: 私有方法 `generateSuggestions` 被路由层调用导致类型检查失败

* **涉及文件**:
  - `src/app/api/search/suggestions/route.ts`
  - `src/lib/search/search-service.ts`
* **问题现象**:
  - 报错：`Property 'generateSuggestions' is private and only accessible within class 'SearchService'.`
  - 出错代码：`SearchService.generateSuggestions(query)`
* **根因分析**:
  - `SearchService.generateSuggestions` 在 service 层被声明为 `private static`
  - 但路由层需要直接调用该能力以返回搜索建议，导致访问修饰符冲突
* **修复方案**:
  - 将 `generateSuggestions` 从 `private static` 调整为 `static`（public），使其成为可复用的服务能力
  - 保持实现逻辑不变（当前返回空数组，后续可按 TODO 扩展为基于历史/热门的建议）
* **修复结果**:
  - `next build` 类型检查阶段不再被该处阻断
  - 搜索建议 API 可正常编译与调用
