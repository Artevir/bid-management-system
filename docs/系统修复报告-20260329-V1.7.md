# 系统修复报告-20260329-V1.7

**版本**: V1.7  
**日期**: 2026-03-29  

## 修复背景

服务器执行 `pnpm build` 在 TypeScript 类型校验阶段失败，阻断生产构建。

## Issue 1: `bidService.createDocument` 返回结构与调用方使用方式不一致

* **所在文件**: `src/app/(app)/bid/page.tsx`
* **问题现象**:
  - 报错：`Property 'data' does not exist on type '{ documentId: number; }'.`
  - 出错代码：`router.push(\`/bid/${res.data.documentId}/edit\`);`
* **根因分析**:
  - `bidService.createDocument` 的实现为：`api.post(...).then(res => res.data)`
  - 因此调用方拿到的 `res` 结构是 `{ documentId: number }`，并不存在 `res.data`
* **修复方案**:
  - 将调用方改为使用 `res.documentId`：
    - `router.push(\`/bid/${res.documentId}/edit\`);`
* **修复结果**:
  - 类型检查阶段不再因该处阻断构建
  - 创建文档后跳转路径正确
